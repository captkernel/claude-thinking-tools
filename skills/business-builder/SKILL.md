# Business builder (guided chain)

Run the user from idea to first customers, ONE step at a time, using each step's output as the next step's input. Don't dump everything at once.
1. Pressure-test the idea for fatal flaws (YC style).
2. Validate the problem: painkiller vs vitamin; where do these people already complain, in what words.
3. Design the offer with the Hormozi value equation (dream outcome x likelihood / time x effort), with risk reversal.
4. Write the sales page with AIDA, built hook -> bullets -> full page.
5. Plan 7 days of content that each drives traffic or builds trust.
6. Find the first 10 customers manually (do things that don't scale).
A "no" at step 1-2 is a win. Ask for the idea and target customer to begin.
